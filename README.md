#  Mjolnir — 3D CAD Design

A 3D CAD model of **Mjolnir**, the legendary hammer of Thor from Norse mythology (and the Marvel universe). Designed as a personal CAD project to practice modeling intricate geometry, engravings, and proportions.

## 👀 How to view the design

### Option 1: View in your browser (easiest)
Just click on **[`mjolnir.stl`](mjolnir.stl)** — GitHub will render it as an interactive 3D model. You can:
-  **Rotate** by clicking and dragging
-  **Zoom** with your scroll wheel
-  **Pan** by right-clicking and dragging

### Option 2: Open in CAD software
Download the **`mjolnir.step`** file and open it in:
- SolidWorks
- Autodesk Fusion 360
- Autodesk Inventor
- FreeCAD (free & open-source)
- Onshape (free in browser)
- Or any other CAD software that supports STEP files

##  Design details

- **Software used:** Fusion 360 
- **Use case:** 3D printing, rendering, animation, or as a CAD learning reference

---

##  3D printing notes (optional)

If you'd like to 3D print this design:
- Recommended layer height: 0.2 mm
- Suggested infill: 20–30%
- Print orientation: handle vertical for best surface finish
- Supports: required under the hammer head
